## Extract
| Command | Description |
| ------- | ----------- |
| `tar -xvzf file.tar.gz` | `x` to extract a file, `v` to do it verbosoly, `z` to use gzip as the extraction method, `f` this must be the last flag of the command, tells the next argument is gonna me a file |
